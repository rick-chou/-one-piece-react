## Show

### When To Use

> equivalent to `condition ? <Component-A /> : <Component-B />`

### Example

<Example col={2} />

### API

| Property     | Description                               | Type              | Default |
| ------------ | ----------------------------------------- | ----------------- | ------- |
| **when**     | condition                                 | `boolean`         | -       |
| **fallback** | show the UI when the condition is not met | `React.ReactNode` | -       |
