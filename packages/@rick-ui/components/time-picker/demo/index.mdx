### API

| Property         | Description                       | Type                                           | Default     |
| ---------------- | --------------------------------- | ---------------------------------------------- | ----------- |
| **currentTime**  | current selected time             | `[number, number]`                             | -           |
| **defaultTime**  | default time                      | `[number, number]`                             | currentTime |
| **onChange**     | time change callback              | `(startTime: number, endTime: number) => void` | -           |
| **refreshToken** | reRender when refreshToken change | `any`                                          | -           |
| **name**         |                                   | `string`                                       | Time        |
